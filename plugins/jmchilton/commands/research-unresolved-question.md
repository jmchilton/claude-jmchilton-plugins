You're going to be give a research note or implementation plan with a list of unresolved questions, a question number or description, and (optionally) some additional note or advice.

Launch a subagent to do the following:

Read in the original document and find the unresolved question. Do lots of research to determine a good answer
for the question and prepare a Markdown report with a
detailed description of the research question.

Write the Markdown report to the same directory as the 
original research or plan - give it a similar name but
append a suffix to the basename to give a indication
about what is being clarified, resolved, etc..
