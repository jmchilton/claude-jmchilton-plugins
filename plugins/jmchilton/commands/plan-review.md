The user will supply you with a plan file that
has not been implemented and optionally a file that
describes the larger goal the plan is part of or a larger
plan that the plan is a sub-step of.

Launch a subagent to review the plan for clarity
and correctness. If you're given a second document also
evaluate the plan in context of that larger plan or particular
goal. Does this work seem to be well motivated and a valid
step toward the larger plan or goal.

Come up with a list of prioritized potential next steps
if any.

Have the subagent give you the summary of this research
and then you should write this file in the same directory as the
plan with _REVIEW appended to the base name of the supplied plan file.

After you've written out the plan, summarize the big points to the user
and then use the AskUserQuestion tool to prompt the user what to work on next.
